دخول المشرف

الدخول عبر إيميل/كلمة مرور (Supabase Auth). بعد الدخول، النظام يتأكد أنك ضمن جدول admins.

أول مرة: من Supabase → Authentication → Users → Add user ثم أضيفي UID في جدول admins.

الطلبات

بحث + فرز حسب التخصص/المدينة/الحالة. تنزيل الـ CV يتم برابط مؤقت (Signed URL) لأنه Bucket خاص.
